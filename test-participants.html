QUBIC Raffle (Standard)

Token Raffle

How to Use:

  1. QUBIC Raffle: Enter an epoch number and click "Load Participants" to see all wallets that participated in the standard QUBIC raffle
  2. Token Raffle: Enter an epoch number and token raffle index (0, 1, 2, etc.) to see participants for a specific token raffle
  3. The table shows each participant's wallet address, number of entries, and percentage of total entries

Note: This queries transaction history from the Qubic network, so it may take a few seconds to load.
